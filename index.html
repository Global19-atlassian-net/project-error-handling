Error Handling Project Group

Welcome to the repository for the Error Handling Project Group! We're focused on defining and communicating Rust's error handling story.

This is the repository we use to organise our work. Please refer to our charter for more information on our goals and current scope.

How Can I Get Involved?

Participation in this project group is open to all. You can see our work in progress via our open issues on this repository and you can participate in discussions via our Zulip stream. We primarily coordinate via our meetings, which are tracked [here][meeting-schedule]. If you're interested in attending our meetings but can't due to a schedule conflist please let us know on the scheduling issue so we can best accommodate everyone.
